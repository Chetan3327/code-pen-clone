## Installation

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/Chetan3327/code-pen-clone.git
   ```
2. Navigate to the project directory:

   ```bash
   cd code-pen-clone
   ```
3. Install dependencies:

   ```bash
   npm install
   ```
